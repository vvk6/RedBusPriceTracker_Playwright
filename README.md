# RedBusPriceTracker_Playwright
Playwright Price Tracker 🚌 A TypeScript-based automation tool that scrapes travel data, handles dynamic overlays, and sends price alerts via Telegram. Scheduled via GitHub Actions to provide hands-on experience with CI/CD and modern web-testing architecture.


🚀 Project Purpose & Evolution
This project started as a utility to solve a personal pain point: tracking fluctuating bus fares. While commercial anti-bot measures and frequent UI updates make long-term scraping a challenge, this project served as a technical deep-dive into the Playwright ecosystem.

What I achieved:

Selenium to Playwright Migration: Transitioned from Java/Selenium to TypeScript/Playwright.

Infrastructure as Code: Set up a self-sustaining tracker on GitHub Actions.

Observability: Integrated a Telegram Bot to push logs and visual snapshots of the UI state.

Problem Solving: Resolved complex "Strict Mode" and "Actionability" issues inherent in modern SPA (Single Page Application) testing.
